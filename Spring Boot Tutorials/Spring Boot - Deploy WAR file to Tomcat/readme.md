<p>In this article, we will show you how to deploy a Spring Boot WAR file to the Tomcat servlet container.</p>

<p>For Spring Boot WAR deployment, you need to do three steps:</p>

<ol>
	<li>Extends SpringBootServletInitializer</li>
	<li>Marked the embedded servlet container as provided.</li>
	<li>Update packaging to war</li>
</ol>

<p><em>Tested</em></p>

<ol>
	<li>Spring Boot 1.4.2.RELEASE</li>
	<li>Tomcat 8.5.9</li>
	<li>Maven 3</li>
</ol>
