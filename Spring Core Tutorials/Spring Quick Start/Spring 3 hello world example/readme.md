This tutorial shows you how to create a simple hello world example in Spring 3.0.

Technologies used in this article :

1.  Spring 3.0.5.RELEASE
2.  Maven 3.0.3
3.  Eclipse 3.6
4.  JDK 1.6.0.13

_P.S Spring 3.0, at least JDK 1.5 is required to work._
