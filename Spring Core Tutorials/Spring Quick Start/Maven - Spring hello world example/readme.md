This quick guide example uses Maven to generate a simple Java project structure, and demonstrates how to retrieve Spring bean and prints a “hello world” string.

Technologies used in this article :

1.  Spring 2.5.6
2.  Maven 3.0.3
3.  Eclipse 3.6
4.  JDK 1.6.0.13
